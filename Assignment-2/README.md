How I Debugged My Program

Today, I worked on improving my console-based application by restructuring the logic into separate methods with proper parameters and return values. This made the code more organized and easier to manage.

Enhancements I Made:

- Implemented input validation using "TryParse" to avoid invalid user inputs
- Added exception handling using "try-catch" blocks to handle runtime errors

Debugging Process:

- Used breakpoints to stop execution at key points in the code
- Monitored variable values (like choice, number, result) using the Watch window
- Applied Step Over (F10) to execute code line by line
- Used Step Into (F11) to go inside methods and understand internal logic

Outcome:

By debugging step-by-step, I was able to identify logical errors quickly and ensure that each method was working correctly. This process improved both my coding and problem-solving skills.
