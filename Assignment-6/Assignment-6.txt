1. Git rebase

• Used to apply commits on top of another branch.
• Example :- git rebase main

2. Git merge

• Used to combine two branches.
• Example :- git merge feature-branch

3. Git reset

• Used to undo commits or changes.
• Example :- git reset --hard HEAD~1

4. Git restore

• Used to restore files to previous state.
• Example :- git restore file.txt
